# Installation

s1engine can be used as a Rust library, WASM npm package, or Docker container.

## Prerequisites

| Platform | Requirements |
|----------|-------------|
| Rust | Rust 1.88+ via [rustup](https://rustup.rs/) |
| WASM | Node.js 18+, `wasm-pack` |
| Docker | Docker 20+ |

## Choose Your Path

- **[Rust Library](./rust.md)** — Server-side document processing
- **[npm / WASM](./npm.md)** — Browser-based editing
- **[Docker](./docker.md)** — Full editor, zero setup
